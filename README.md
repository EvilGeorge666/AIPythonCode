# AIPythonCode

A small top-down, grid-based Python game where you control a roomba and clean up trash while avoiding poop.

## Game Objective
- Move the roomba around the grid.
- Clean all randomly placed trash tiles.
- Avoid poop that randomly spawns over time.
- Touching poop ends the game.

## Run
```bash
python3 roomba_game.py
```

## Controls
- Arrow keys or `WASD` to move
- `R` to restart after win/loss
